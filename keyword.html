Mission 03 // Day 3

Keyword Cipher

The most powerful cipher yet — because the key isn't a number, it's a whole codebook. Learn to parse it, search it, and use it to swap words in a message.

The Codebook Concept

The Caesar cipher shifts every letter by the same amount — anyone who knows the rule and the shift can break it. The Keyword cipher is different: instead of shifting letters, you swap entire words using a secret codebook.

The codebook is a list of pairs: real word,code word. To encode a message, you look up each word. If you find it, replace it. If you don't, leave it as-is.

dog,bravo
cat,alpha
run,delta
meet,foxtrot
tonight,sierra
at,tango
the,november
park,lima

Example encoding:

"Meet the dog at the park"
→
"foxtrot november bravo tango november lima"

The codebook is the secret key. Two people can send "foxtrot november bravo tango november lima" and anyone intercepting it just sees words — unless they have the codebook.

To decode: do the same lookup in reverse. Find the code word in the right column, replace it with the real word in the left column.

Java Concepts You'll Use

split(",") — Different Delimiter

You already used split(" ") to split on spaces. You can split on any character — the codebook uses commas.

String entry = "dog,bravo";
String[] parts = entry.split(",");
// parts[0] is "dog"   — the real word
// parts[1] is "bravo" — the code word
dog parts[0]
bravo parts[1]

equals() — The Right Way to Compare Strings

Here's a trap almost every Java beginner falls into: using == to compare Strings.

String word = "hello";

// WRONG — this often fails even when strings look identical
if (word == "hello") { ... }

// RIGHT — always use .equals() for String comparison
if (word.equals("hello")) { ... }

== checks if two variables point to the exact same object in memory. .equals() checks if they contain the same characters. For Strings, you almost always want .equals(). Just make it a habit — it will save you hours of confusing bugs.

The Two-Level Split Pattern

Here's the core algorithm, step by step:

// Encoding "meet the dog"

1. Split sentence into words:

meet
the
dog

2. For each word, loop through every codebook entry:

Checking "meet"...

Split "dog,bravo" → [dog][bravo] → "meet".equals("dog")? No

Split "meet,foxtrot" → [meet][foxtrot] → "meet".equals("meet")? Yes!

Use "foxtrot" → add to result

3. Words not in codebook pass through unchanged.

Pass-Through Logic

What happens if a word isn't in the codebook? Nothing — you just keep the original word.

String encodedWord = word;   // default: unchanged

for (String entry : codebook) {
    String[] parts = entry.split(",");  // split into [real, code]
    if (word.equals(parts[0])) {        // found a match?
        encodedWord = parts[1];         // use the code word
        break;                          // no need to keep searching
    }
}
// encodedWord is now either the swap or the original — both cases handled

Your Task

Your teacher will give you two ArrayLists: the lines of a message, and the codebook entries already loaded from a file. Write the encode and decode methods.

// Encodes a message using the codebook
// lines:    each element is one line of the plaintext message
// codebook: each element is a String like "dog,bravo"
public ArrayList<String> encode(ArrayList<String> lines, ArrayList<String> codebook) {

    ArrayList<String> result = new ArrayList<String>();

    for (String line : lines) {

        // Split this line into individual words
        String[] words = line.split(" ");
        String encodedLine = "";

        for (String word : words) {
            String encodedWord = word;   // default: pass through unchanged

            // Search the codebook for a matching entry
            for (String entry : codebook) {

                // Split the entry into its two parts: [realWord, codeWord]
                String[] parts = entry.split(",");

                // Is this the word we're looking for?
                if (word.equals(parts[0])) {
                    encodedWord = parts[1];  // use the code word instead
                    break;                   // stop searching — we found it
                }
            }

            // Add encoded word to the line (with a space separator)
            if (!encodedLine.isEmpty()) encodedLine += " ";
            encodedLine += encodedWord;
        }

        result.add(encodedLine);
    }

    return result;
}

// Decodes a message (reverse lookup — find code word, return real word)
// Identical logic, just swap which part you compare and which part you return
public ArrayList<String> decode(ArrayList<String> lines, ArrayList<String> codebook) {

    ArrayList<String> result = new ArrayList<String>();

    for (String line : lines) {
        String[] words = line.split(" ");
        String decodedLine = "";

        for (String word : words) {
            String decodedWord = word;   // default: unchanged

            for (String entry : codebook) {
                String[] parts = entry.split(",");
                // TODO: this time, match against parts[1] (the code word)
                if (word.equals(parts[1])) {
                    decodedWord = parts[0];  // return the real word
                    break;
                }
            }

            if (!decodedLine.isEmpty()) decodedLine += " ";
            decodedLine += decodedWord;
        }

        result.add(decodedLine);
    }

    return result;
}
