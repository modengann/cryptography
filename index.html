THE CRYPTOGRAPHER'S TOOLKIT

// A THREE-DAY MISSION IN STRING MANIPULATION

Cryptography is the science of hiding information. For thousands of years, sending a secret message was a matter of life and death. Today you're going to learn how it works — and build your own encoder in Java.

What Is Cryptography?

Every time you log into a website, every time your phone connects to WiFi, cryptography is running in the background — turning your data into scrambled nonsense that only the right key can unlock. Without it, anyone on the same network could read your passwords in plain text.

In this project you'll build three different ciphers, each one more powerful than the last. Each cipher also teaches you a new Java skill: string splitting, char arithmetic, and data lookup from a codebook.

By Day 3 you'll have a working multi-layer encoder. Not the kind that secures bank transactions — but built on the exact same fundamental ideas.

What You'll Build

Over three days you'll write Java methods that can take a plain-text message and run it through multiple layers of encryption. Here's the full pipeline:

Your teacher will supply the text files and test your output against a known answer. Your job is to write the transformation logic for each step — the interesting part.

And when you're done: the encoded output looks like complete nonsense to anyone without your cipher chain. That's exactly the goal.
