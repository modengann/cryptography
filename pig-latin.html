Mission 01 // Day 1

Pig Latin Encoder

Your first cipher. Simple, but it requires real string manipulation — split, charAt, substring. Let's break it down step by step.

What Is Pig Latin?

Pig Latin is a made-up language where you take each word, move its first letter to the end, and add "ay". It's not a secure cipher, but it's a perfect first exercise — it forces you to pull apart a String and put it back together in a different order.

The same three steps work for every word:

Example: "hello" → "ellohay"

Step 1: Take the first letter —
'h'
← that's charAt(0)
Step 2: Take everything after it —
"ello"
← that's substring(1)
Step 3: Reassemble —
"ello"
+
"h"
+
"ay"
→
"ellohay"

Another example: "cryptography" → "ryptographycay". That's it — three steps, every word. Easy to describe, genuinely useful to code.

Java Methods You'll Use

split(" ")

When you call split(" ") on a String, Java cuts it up at every space and gives you back an array of the individual words.

String sentence = "hello world";
String[] words = sentence.split(" ");
// words[0] is "hello"
// words[1] is "world"
// words.length is 2
hello [0]
world [1]

charAt(0)

Every character in a String has a numbered position. The first one is always index 0. charAt(0) reaches into position 0 and gives you that character.

Think of a String as a row of labeled boxes — here's "hello":

charAt(0) returns 'h' — the character in box 0.

substring(1)

substring(1) gives you all the characters starting from index 1 — which is everything except the first letter. Think of it as "skip box 0, give me the rest."

substring(1) returns "ello" — the dimmed letter is skipped.
